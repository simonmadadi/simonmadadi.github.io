Smuci Mobile App

Interface Design / Mobile Application Design

Project Overview

For this project I was tasked with designing a user interface for a music application. This project was intended to focus on user goals as the starting point for the research.

Process

I began this project by conducting research surrounding music applications. I created a survey with a group of 7 music application users that gave me a rough understanding of their behavioral variables. I then continued my research by creating personas surrounding users for this application. This step allowed me to get a better understanding of who I would be designing for.




After conducting my research I then went on to create wireframes of the application's interface. It was at this stage that I realized the limitations of the amount of information that could be fit onto a mobile phone display, so comprimises had to be made in order to keep the usability.

I decided to create a navigational portion on the bottom half of the screen that would always appear to user so long as a playlist was selected. I also wanted it to be easy for users to pick a new playlist while listening to their current one so I allowed for space at the top of the screen for the rest of the app to function normally. I then finalized the graphical interface of the application after revising the wireframes.

The Warriors Kinetic Typography

Motion Graphics

Project Overview

For this project I was given the task of taking a selected audio clip and creating a compeling video around the dialogue using typography and some symbols. I chose to create a video based off the film "The Warriors" because I've always enjoyed the dialogue fro the movie.

Challenges

Creating a video in After Effects for the first time was definitely a challenge, but once I understood the interface better it became a lot more effortless. I also faced the challenge of the audience cheering in the audio clip and how I would go about representing that on screen. I solved this by having the cheers drown out the dialogue from the main character.

Russian Fox Experiments

Motion Graphics

Project Overview

This project gave me the task of creating an informational animation restricted to using simple geometry only. I knew I wanted to focus on this topic after reading about Dmitry Belyaev and his experiments. I wanted to make an easy to understand video that explained a relatively complex topic while also making it interesting to the audience.

Process

I started this project by writing a script to outline what would be said in the video. I then created a storyboard to get a better understanding of what would be on screen at any given moment. I created my images using Illustrator and imported them into After Effects to complete the animation.

Problem Solved

Book Cover Design

Project Overview

For this project I was tasked with redesigning a cover for the book Problem Solved. I wanted to create something radically different from the original.

Process

I began this project by sketching several thumbnails that pertained to the title. I used the images that I immediatly associated with problem solving. My focus for these concepts was on the idea of solving a physical problem. I explored the idea of a problem being several intertwined tasks. After trying out a few ideas I realized that I wanted to focus my design around maze imagery. My final design uses a maze to illustrate a problem as well as the shift in mind set that is illustrated through the change in colour.
